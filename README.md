Project Introduction

SmartTrain is a full-stack web-based train reservation system. It solves the problem of complex, outdated train booking systems by providing a modern Single Page Application where users can search trains between any two Turkish cities, select specific seats on an interactive seat map, make simulated payments, and manage their bookings — all secured with JWT authentication. An admin panel provides real-time stats, booking oversight, and train management.
