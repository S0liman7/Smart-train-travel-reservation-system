## Project Introduction

A web-based platform for train travel that allows users to register and search for trains between Turkish cities by selecting origin, destination, date, class, and number of passengers. The system supports one-way and round-trip searches, interactive seat selection, simulated payment, and booking management. An admin panel provides full oversight of users, bookings, and train schedules.
